# Projeto de Programação para Web 1 - Biblioteca Musical

## Descrição do Projeto

Este projeto tem como objetivo desenvolver uma aplicação web onde usuários podem registrar suas notas e opiniões sobre músicas ouvidas. Nele, será utilizada a tecnologia de HTML, CSS e Python com Flask, além de CSV para permanência de dados. Foi desenvolvido para o projeto final de Programação para Web1 do 1° período do curso de Engenharia de Software 

Na aplicação o usuário poderá favoritar, comentar e curtir suas músicas favoritas, editar suas informações de perfil, visualizar os álbuns favoritados e comentados e retirar os mesmos de suas seções específicas. Além disso, os administradores da aplicação podem cadastrar novos álbuns por meio de formulários na própria interface.
